# Multi-forest (Letsplant)

## Overview
**Multi-forest** is a real-time, multi-user web interaction experiment. It explores how "collective will" shapes digital landscapes. Through long-press interactions, all online participants collaboratively decide the growth, reproduction, and evolution of a shared digital ecosystem.

## Core Concepts
* **Decentralized Agency:** The system does not obey a single user; it responds to the emergent behavior of the group.
* **Devolution of Power:** The creator steps back, allowing the system to form its own ecology through multi-user intervention.
* **Shared Resilience:** The survival of the forest depends on silent cooperation between strangers.

## Technical Stack
* **Backend:** Node.js, Socket.io (Real-time data synchronization)
* **Frontend:** HTML5 Canvas / p5.js, CSS3
* **Logic:** Persistent state management for plant growth algorithms.
